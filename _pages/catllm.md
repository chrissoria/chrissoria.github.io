## Python Apps

<a href="https://pypi.org/project/cat-llm/">Classify Survey Responses, Images, and PDF Files</a>: CatLLM is a tool for categorizing, extracting categories, and scoring text and images. Contains custom-functions for scoring the CERAD Constructional Praxis test for dementia classification. <a href="https://github.com/chrissoria/cat-llm">[GitHub]</a>

<a href="https://pepy.tech/projects/cat-llm"><img src="https://static.pepy.tech/badge/cat-llm/month" alt="Downloads per month"></a> <a href="https://pepy.tech/projects/cat-llm"><img src="https://static.pepy.tech/badge/cat-llm" alt="Total Downloads"></a>

<a href="https://pypi.org/project/llm-web-research/">Build Datasets from Web Data</a>: Part of the CatLLM ecosystem, the llm-web-research package enables LLM-powered web research with a focus on accuracy over quantity. Uses a multi-step verification pipeline to catch ambiguous queries before returning potentially incorrect answers. <a href="https://github.com/chrissoria/llm-web-research">[GitHub]</a>
