## Python Apps

<a href="https://pypi.org/project/cat-llm/">CatLLM</a>: A tool for categorizing, extracting categories, and scoring text and images. Contains custom-functions for scoring the CERAD Constructional Praxis test for dementia classification.

<a href="https://pypi.org/project/llm-web-research/">Build Datasets from Web Data</a>: Part of the CatLLM ecosystem, the llm-web-research package enables LLM-powered web research with a focus on accuracy over quantity. Uses a multi-step verification pipeline to catch ambiguous queries before returning potentially incorrect answers.
